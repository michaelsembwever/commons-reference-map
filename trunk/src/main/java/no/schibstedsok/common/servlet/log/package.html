This package provides a supporting servlet, LoggingServlet, that provides a listing of all Log4J loggers active with their current threshold level.
It also allows modifying these levels at runtime.

Usage of this servlet requires the addition to web.xml of 

    <servlet>
        <servlet-name>Log</servlet-name>
        <servlet-class>no.schibstedsok.common.servlet.log.LoggingServlet</servlet-class>
    </servlet>
    
    <servlet-mapping>
        <servlet-name>Log</servlet-name>
        <url-pattern>/Log</url-pattern>
    </servlet-mapping>
